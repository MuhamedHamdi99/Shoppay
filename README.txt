E-commerce application, built with Spring Boot
, is a user-friendly platform that allows customers to browse and purchase products online.

Key features of our application include:

1- Product Catalog: Customers can explore our wide range of products through an organized catalog. Each product listing provides essential details and images.

2- Shopping Cart and Checkout: Customers can add products to their cart and proceed to a simple and secure checkout process. They can review their cart, select shipping options, enter billing information, and complete their purchase.

3- User Accounts: Customers can create accounts to manage their profile information.

4- Order Management: Administrators can efficiently manage incoming orders, update order statuses, and generate invoices.

5- Easy Product Management: Administrators can easily add new products, update inventory quantities, and manage product variations.

6-  This application simplifies the online shopping process, allowing customers to find and purchase products with ease. It provides a secure and user-friendly platform for businesses to sell their products effectively.




PROJECT LAUNCH STEPS:
------------------------------------------
1- Download the Shoppay project zip file and extract it.
2- Install XAMPP and open it. Start Apache and MySQL services.
3- Open Spring Tool Suite 4 IDE or Eclipse.
4- Navigate to "File" > "Open Project from File System". Click on the directory and choose the project folder. Click "Finish" to proceed.
5- Run the project by right-clicking on the project folder and selecting "Run As" > "Spring Boot App".
6- Open any web browser and enter the URL "http://localhost:8085/". Press Enter.
7- Register as a new customer or log in to the project to browse its features.

"PLEASE NOTE" that these steps assume familiarity with XAMPP, Spring Tool Suite 4 IDE or Eclipse, and basic project setup procedures.
